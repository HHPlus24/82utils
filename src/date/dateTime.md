## 82Utils / DATETIME

날짜, 시간과 관련된 Utils를 모아 놨습니다.
- 한국 시간으로 변경하기
- 임의의 날짜에 연,월,일,시간,분,초 더하기
- 임의의 날짜에 연,월,일,시간,분,초 빼기
- 두 날짜의 차이를 일 단위로 계산
- 두 날짜의 사이의 일,주, 월 단위 날짜를 배열 생성
- '-' 형태로 만들기
- '/' 형태로 만들기

## Example
```js

import date from '82utils';

const date: Date = new Date('2023-11-24')
date.toKSTString(date)

```
