# 82utils
---

## 개요

82Utils는 utils 함수들을 만들어서 모아놓은 라이브러리 입니다.

82Utils 패키지는 사소하지만 귀찮은 기능들을 직접 만들어 놓았습니다.

이런 분이 사용하면 좋습니다.

- 팀에서 동일한 Utils 라이브러리를 사용할 필요성이 있을떄
- 사소하지만 필요한 함수들이지만 만들기 귀찮을때
- 다양한 validation 기능이 필요할때
- 필요한 정규표현식을 만들어야 할때
- 시간 / 날짜 관련 함수들과 다양한 Format이 필요 할때
---

## Example
```bash
npm install 82utils
```
